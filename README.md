# Sanskrit & Tamil Tools

Браузерный инструмент для работы с санскритом и тамильским языком: транслитерация, перевод, словарь, грамматика и распознавание текста (OCR). Всё, кроме перевода, работает **локально в браузере** — без отправки данных на сервер.

Приложение — это один HTML-файл со встроенными стилями и шрифтом; рядом лежит папка `sanskrit-tools-assets/` с локальными библиотеками (Tesseract.js, PDF.js) и языковыми данными.

---

## Возможности

### 1. Транслитерация
Преобразование между системами письма:
- **Деванагари ⇄ IAST** (международный алфавит с диакритикой: ā ī ū ṛ ṝ ḷ ṃ ḥ ś ṣ ṭ ḍ ṇ ñ ṅ)
- **Тамильский ⇄ латиница** (ISO 15919)
- **Деванагари ⇄ Harvard-Kyoto** (A I U R RR lR M H z S T D N J G)

Есть экранная клавиатура деванагари, кнопка «Поменять» местами направление и быстрая отправка результата во вкладку перевода.

### 2. Перевод
Машинный перевод через **Anthropic Claude** (модель `claude-haiku-4-5`) прямо из браузера.
- Языки источника: санскрит, тамильский, английский, немецкий, русский.
- Языки перевода: русский, английский, немецкий.
- Требуется API-ключ Anthropic (вводится в шапке, хранится только в вашем браузере в `localStorage`). Получить ключ: [console.anthropic.com](https://console.anthropic.com/settings/keys) (нужно пополнение баланса).

### 3. Параллельный режим
Показывает одновременно три колонки: **оригинал · транслитерация · перевод** — удобно для разбора шлок и стихов.

### 4. Словарь
- **Monier-Williams (санскрит)** — встроенный мини-словарь ключевых терминов (dharma, karma, yoga, brahman, ātman, mokṣa, māyā и др.). Поиск в IAST или деванагари.
- **Тамильский лексикон** — встроенный набор слов (அன்பு, சிவம், ஆன்மா …). Поиск тамильским шрифтом или латиницей.

### 5. Грамматика (санскрит)
Готовые таблицы форм:
- Склонения: a-основа (муж./ср.), ā-основа (жен.), i-, u- и an-основы.
- Спряжения глаголов 1 класса (Парасмайпада и Атманепада).

Введите свою основу — формы достроятся автоматически.

### 6. OCR — распознавание текста
Извлечение текста из **изображений и PDF** (в том числе отсканированных, без текстового слоя) локально, через Tesseract.js.
- Языки: санскрит (деванагари), хинди, тамильский, английский.
- **Двуязычные режимы**: «Санскрит + Английский», «Хинди + Английский», «Тамильский + Английский» — деванагари и латиница распознаются за один проход, в том числе вперемешку.
- Многостраничные PDF обрабатываются по странице; результат выводится единым текстом с разделителями страниц, его можно скопировать или отправить в транслитерацию/перевод.

### 7. Файлы
Загрузка **TXT, Markdown и PDF**. Из PDF извлекается готовый текстовый слой (для сканов без текста используйте вкладку OCR).

### 8. История
Последние операции (транслитерация, перевод, OCR) сохраняются локально в браузере.

---

## Конфиденциальность
Транслитерация, словарь, грамматика, OCR и работа с файлами выполняются полностью в вашем браузере и никуда не передаются. Единственное исключение — машинный перевод: текст отправляется напрямую на серверы Anthropic (Claude). API-ключ хранится только в вашем браузере и не передаётся разработчику.

---

## Запуск локально
Приложению нужны локальные файлы из `sanskrit-tools-assets/` (рабочий поток OCR и PDF), поэтому открывать его лучше через локальный веб-сервер, а не как файл напрямую:

```bash
python3 -m http.server 8000
```
Затем откройте `http://localhost:8000/` в браузере.

На GitHub Pages приложение работает без дополнительных действий.
